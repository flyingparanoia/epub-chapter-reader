# EPUB Chapter Reader

Open a local `.epub` file and render the book as a vertical stack of chapter pages.

## Run

```bash
npm install
npm run dev
```

## Deploy

Push to the `main` branch and GitHub Actions will build and deploy the site to GitHub Pages automatically.

## Features

- Local EPUB upload and drag-and-drop
- Table of contents on the left
- One chapter card per spine document
- Continuous vertical reading flow
- Click-to-jump chapter navigation
- Adjustable reading scale
